Color Inversion

The Negative Effect

Takes the color of every pixel and calculates the inverted color.
Every pixel is formed of red(R), green(G) and blue(B) which have values between 0 and 255, so the inverse will be: R = 255 - R G = 255 - G B = 255 - B

Greyscale

Greyscale with average

Every pixel is formed of red(R), green(G) and blue(B) so for the average method the new R G B values are calculated by calculating the average of the 3 colors. greyscale = ( R + G + B ) ÷ 3

Greyscale with luminance method

The luminosity method uses a formula that takes into accont human sensitivity to different wavelengths. By using this formula the result is more vivid as it preserves the luminosity of the image. greyscale = ( 0.3 × R ) + ( 0.59 × G ) + ( 0.11 × B )

Threshold

Creates a binary image. For this example the pixels were set to be either back or white if the value was lower or greater then the threshold.

Transformations

Transformation Mirror

Moves the pixels in the image to result into the mirror image of the initial picture.

Transformation Upside Down

Moves the pixels in the image to result into the upside down image of the initial picture.

Convolution

Convolution is the process of adding each element to it's local neighbours wheighted by a kernel. Based on the type of kernel used, different effects can be obtained.
