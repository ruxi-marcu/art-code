Sample Game

This is a short sample game with some basic functionality. It is still a work in progress :)

  • Has a loaded model that can be controlled.
  • It uses a state machine where different animations from maximo were added.
  • The character takes the gun when overlap with the gun is detected.
  • The idle animation state changes if character is equiped with the gun or not.
  • Can change from third person to first person.
  • The character can shoot the targets and the first person view can be used for aiming.
  • The project was made using C++ for the classes and a blueprint for the animation of the caracter.
  • Added menus with Widget blueprint
  • Added animations as level sequences
  • Added some sounds
